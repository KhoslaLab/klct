# Update klct

This is a convenience function wrapper around
[`remotes::install_github()`](https://remotes.r-lib.org/reference/install_github.html)
to install/update `klct`. Meant to be used as an RStudio addin.

## Usage

``` r
update_klct()
```

## Examples

``` r
if (FALSE) {
update_klct()
}
```
