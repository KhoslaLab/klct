# Render an R Markdown document with colored text

This function is a wrapper around
[`rmarkdown::render()`](https://pkgs.rstudio.com/rmarkdown/reference/render.html)
which supplies the Lua Pandoc filter from the
[mayodown](https://rdrr.io/pkg/mayodown/man/mayohtml.html) package to
color text. It is used as the `knit` function in the standard (non-Mayo)
R Markdown templates.

## Usage

``` r
knit_with_colored_text(input, ...)
```

## Arguments

- input:

  The input file to be rendered.

- ...:

  Additional arguments passed on to
  [`rmarkdown::render()`](https://pkgs.rstudio.com/rmarkdown/reference/render.html).

## Value

As for
[`rmarkdown::render()`](https://pkgs.rstudio.com/rmarkdown/reference/render.html),
when `run_pandoc = TRUE`, the compiled document is written into the
output file, and the path of the output file is returned. When
`run_pandoc = FALSE`, the path of the Markdown output file, with
attributes `knit_meta` (the knitr meta data collected from code chunks)
and `intermediates` (the intermediate files/directories generated by
[`rmarkdown::render()`](https://pkgs.rstudio.com/rmarkdown/reference/render.html)).

## Examples

``` r
if (FALSE) {
knit_with_colored_text("example_file.Rmd")
}
```
