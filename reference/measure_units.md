# Units for bone microCT measures

A named character vector mapping column names produced by the NeoScan
scanner to their unit strings (assuming mm-based output). Used
internally by
[`get_measure_label()`](https://khoslalab.github.io/klct/reference/get_measure_label.md)
to build display labels for plots.

## Usage

``` r
measure_units
```

## Format

An object of class `character` of length 31.
