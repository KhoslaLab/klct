# Print bone microCT data

This is a wrapper around
[`DT::datatable()`](https://rdrr.io/pkg/DT/man/datatable.html) to print
out bone microCT data at one site by sex for use in an R Markdown
document. Each sex is displayed as an interactive, sortable, searchable
table.

## Usage

``` r
print_data(data, ...)
```

## Arguments

- data:

  A data frame containing bone microCT data, formatted as is the output
  of
  [`read_trabecular()`](https://khoslalab.github.io/klct/reference/read_trabecular.md)
  or
  [`read_cortical()`](https://khoslalab.github.io/klct/reference/read_cortical.md).

- ...:

  Additional arguments passed on to
  [`DT::datatable()`](https://rdrr.io/pkg/DT/man/datatable.html).

## Value

An
[`htmltools::tagList()`](https://rstudio.github.io/htmltools/reference/tagList.html)
containing one
[`DT::datatable()`](https://rdrr.io/pkg/DT/man/datatable.html) widget
per sex.

## Examples

                              mouse_table, site = "Spine")
print_data(spine_trab)
